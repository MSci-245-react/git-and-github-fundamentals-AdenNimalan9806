# JavaScript REPL Practice Lab 1 - Assignment 1

One thing I learned in Lecture 1 is that JavaScript has some strange behaviours, such as `typeof null` which returns `"object"` even though this is not the correct type for null.

## Things I learned in Lecture 1:

- JavaScript web development evolved from static sites to dynamic, full-stack applications.
- Single Page Applications update the content of the site without constantly reloading the page with the updates.

The key `typeof` can be used to check a value's type.

```javascript
typeof null
```

```
object
```

The expression `0.1 + 0.2` demonstrates how floating-point math in JavaScript can sometimes produce unexpected results.

```javascript
0.1 + 0.2
```

```
0.30000000000000004
```
